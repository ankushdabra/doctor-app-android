# VitalSync Doctor - Android Application

VitalSync Doctor is a comprehensive, professional healthcare management application designed for doctors. It streamlines daily clinical workflows, from managing patient appointments to creating digital prescriptions, all within a modern and intuitive user interface.

## âœ¨ Key Features

- **Professional Dashboard**: Real-time stats on patients seen today, total earnings, and upcoming appointment counts.
- **Smart Scheduling**: Efficiently manage patient bookings and view detailed consultation schedules.
- **Digital Prescriptions**: Create and issue digital prescriptions instantly with integrated medicine search and dosage instructions.
- **Patient History**: Access comprehensive patient records and past prescription history for better clinical decision-making.
- **Profile Management**: Complete control over professional details, consultation fees, clinic address, and weekly availability (with AM/PM slot management).
- **Modern UI/UX**: Built with Jetpack Compose following Material 3 guidelines, supporting both Light and Dark modes.
- **Secure Authentication**: JWT-based secure login and registration flow for medical professionals.

## ðŸ›  Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Design System**: [Material 3](https://m3.material.io/)
- **Asynchronous Programming**: [Coroutines](https://kotlinlang.org/docs/coroutines-overview.html) & [Flow](https://kotlinlang.org/docs/flow.html)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Networking**: [Retrofit](https://square.github.io/retrofit/) with OkHttp
- **Local Storage**: [Preferences DataStore](https://developer.android.com/topic/libraries/architecture/datastore)
- **Image Loading**: [Coil](https://coil-kt.github.io/coil/)
- **JSON Parsing**: [Gson](https://github.com/google/gson)
